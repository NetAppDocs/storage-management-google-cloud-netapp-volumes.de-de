=== Unterstützung für Google Cloud NetApp Volumes in BlueXP

Sie können Google Cloud NetApp Volumes jetzt direkt von BlueXP aus verwalten:

* Fügen Sie eine Arbeitsumgebung hinzu.
* Bände anzeigen.
* Arbeitsumgebungen entfernen.
